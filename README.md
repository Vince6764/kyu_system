# KYU Results Management System 🎓
A secure web-based portal developed for **Kirinyaga University** to manage student grades and administrative tasks.

## 🚀 Features
* **Dual Portal System:** Separate login interfaces for Students and Staff (Admins).
* **Secure Authentication:** Uses PHP `password_hash` and `password_verify` for credential security.
* **Interactive UI:** * Eye-toggle visibility for password fields.
    * Real-time progressive password strength meter.
    * SweetAlert2 notifications for a modern look.
* **Database Integration:** Built with PHP PDO to prevent SQL injection.

## 🛠️ Tech Stack
* **Backend:** PHP 8.x
* **Database:** MySQL (MariaDB)
* **Frontend:** HTML5, CSS3 (Water.css framework), JavaScript
* **Tools:** XAMPP, Git, SweetAlert2
